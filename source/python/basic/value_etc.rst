値 もろもろ
========================================

値として同じ？オブジェクトとして同じ？
----------------------------------------

.. code-block:: python

    >>> l1 = [1,2,3,4,5]
    >>> l2 = [1,2,3,4,5]

    >>> id(l1)
    4542998472
    >>> id(l2)
    4541694408

    # 値として同じか
    >>> l1 == l2
    True

    # オブジェクトとして同じか
    >>> l1 is l2
    False

    >>> l1 = l2 = [1, 2, 3, 4, 5]
    >>> (l1)
    4542954504
    >>> id(l2)
    4542954504

    # 値として同じか
    >>> l1 == l2
    True

    # オブジェクトとして同じか
    >>> l1 is l2
    True
