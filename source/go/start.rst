Goを始める
========================================

goスクリプトの拡張子
-----------------------------------

- .go をつける

メインパッケージ
-----------------------------------

- 単独で動作する実行可能なプログラムを定義する

.. code-block:: go

    package main

    func main() {
    
    }


goスクリプトを実行する
-----------------------------------

.. code-block:: bash

    $ go run [goスクリプト]


goスクリプトをビルドする
-----------------------------------

.. code-block:: bash

    $ go build [goスクリプト]
