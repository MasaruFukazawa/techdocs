パッケージ
===================================

- 個々のパッケージは、パッケージ名を持つ
- パッケージの呼び出し方法

.. code-block:: go

    pacage パッケージ名

    import "パッケージ名"
    import (
        "パッケージ名01"
        "パッケージ名02"
        "パッケージ名03"
    )

- インポートしたパッケージは、使わないとだめ。エラーになる
- パッケージ内の初期化
	- 配列の初期化などを行うinit関数
	- 個々のファイル内に記述することができる
	- 初期化は、下位から上位に向かって行われる
		- mainパッケージが、最後に初期化される

初期化
-----------------------------------

.. code-block:: go

    func init() {

    }

パッケージ名に別名をつける
-----------------------------------

.. code-block:: go

    import (
        別名 "パッケージ名01"
    )
